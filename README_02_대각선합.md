# 📐 특정 조건의 2차원 배열 합산

## 📋 문제 설명

2차원 정수 배열 `board`와 정수 `k`가 주어집니다.

**`i + j <= k`를 만족하는 모든 `(i, j)`에 대한 `board[i][j]`의 합**을 return하는 solution 함수를 완성해 주세요.

---

## 📌 제한 사항

- 1 ≤ `board`의 길이 ≤ 100
- 1 ≤ `board[i]`의 길이 ≤ 100
- 0 ≤ `board[i][j]` ≤ 10,000
- 0 ≤ k < 200

---

## 💡 핵심 아이디어

`i + j <= k` 조건을 만족하는 영역은 **좌상단 삼각형 모양**입니다.

```
k = 2 일 때 (✅ = 합산, ❌ = 제외)

       j=0  j=1  j=2  j=3
  i=0 [ ✅   ✅   ✅   ❌ ]   → 0+0=0 ✅  0+1=1 ✅  0+2=2 ✅  0+3=3 ❌
  i=1 [ ✅   ✅   ❌   ❌ ]   → 1+0=1 ✅  1+1=2 ✅  1+2=3 ❌
  i=2 [ ✅   ❌   ❌   ❌ ]   → 2+0=2 ✅  2+1=3 ❌
  i=3 [ ❌   ❌   ❌   ❌ ]   → 3+0=3 ❌
```

---

## ✅ 정답 코드

```cpp
#include <string>
#include <vector>
#include <iostream>
using namespace std;

int solution(vector<vector<int>> board, int k) {
    int answer = 0;
    for (int i = 0; i < board.size(); i++) {
        for (int j = 0; j < board[i].size(); j++) {
            if (i + j <= k) {
                answer += board[i][j];
            }
        }
    }
    return answer;
}

int main() {
    // 테스트용 board 초기화
    vector<vector<int>> board = {
        {1, 2, 3},
        {4, 5, 6},
        {7, 8, 9}
    };
    int k = 2;

    cout << solution(board, k) << endl;  // 결과 출력
    return 0;
}
```

---

## 🔍 코드 분석

### 이중 반복문 구조

```
외부 루프: i = 0 ~ board.size()-1   (행 순회)
내부 루프: j = 0 ~ board[i].size()-1 (열 순회)
  └─ 조건 검사: i + j <= k 이면 합산
```

| 구성 요소 | 설명 |
|-----------|------|
| `board.size()` | 배열의 행 개수 |
| `board[i].size()` | i번째 행의 열 개수 |
| `if (i + j <= k)` | 삼각형 영역 조건 필터 |
| `answer += board[i][j]` | 조건 만족 시 누적 합산 |

## 🧪 테스트 케이스

### 예시 1
```
board = [[1, 2, 3],
         [4, 5, 6],
         [7, 8, 9]], k = 2

합산 위치:
  (0,0)=1  (0,1)=2  (0,2)=3  → 0+0=0✅  0+1=1✅  0+2=2✅
  (1,0)=4  (1,1)=5            → 1+0=1✅  1+1=2✅  1+2=3❌
  (2,0)=7                     → 2+0=2✅  2+1=3❌

결과: 1+2+3+4+5+7 = 22
```

### 예시 2
```
board = [[1, 2],
         [3, 4]], k = 1

합산 위치:
  (0,0)=1  (0,1)=2  → 0+0=0✅  0+1=1✅
  (1,0)=3            → 1+0=1✅  1+1=2❌

결과: 1+2+3 = 6
```

### 테스트 표

| board | k | 결과 |
|-------|---|:----:|
| `[[1,2],[3,4]]` | 0 | **1** |
| `[[1,2],[3,4]]` | 1 | **6** |
| `[[1,2,3],[4,5,6],[7,8,9]]` | 2 | **22** |

## 📊 시간/공간 복잡도

| 항목 | 복잡도 |
|------|--------|
| 시간 복잡도 | O(N × M) |
| 공간 복잡도 | O(1) |

- N: board의 행 수, M: board의 열 수
- 모든 원소를 한 번씩 순회하며 조건 검사
