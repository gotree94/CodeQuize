# 🍖 양꼬치 계산기

## 📋 문제 설명

머쓱이네 양꼬치 가게는 **10인분을 먹으면 음료수 하나를 서비스**로 줍니다.

| 항목 | 가격 |
|------|------|
| 양꼬치 1인분 | 12,000원 |
| 음료수 1개 | 2,000원 |

정수 `n`과 `k`가 매개변수로 주어질 때,  
양꼬치 `n`인분과 음료수 `k`개를 먹었다면 **총 얼마를 지불해야 하는지** return하도록 solution 함수를 완성하세요.

---

## 📌 제한 사항

- 0 < n < 1,000
- 0 < k < 1,000
- 서비스로 받은 음료수는 지불 금액에서 제외

---

## 💡 핵심 아이디어

- 서비스 음료수 개수 = **양꼬치 인분 수 기준** → `n / 10` (정수 나눗셈)
- 실제 지불 음료수 수 = `k - (n / 10)`
- **총 금액** = 양꼬치 비용 + 실제 음료수 비용

```
총 금액 = n × 12,000 + k × 2,000 - (n/10) × 2,000
```

---

## ✅ 정답 코드

```cpp
#include <string>
#include <vector>
#include <iostream>
using namespace std;

int solution(int n, int k) {
    int answer = 0;
    answer = n * 12000 + k * 2000 - (n / 10) * 2000;
    return answer;
}

int main() {
    // 테스트 케이스로 직접 호출
    cout << solution(10, 3) << endl;   // 결과 출력
    return 0;
}
```

---

## 🔍 코드 분석

| 구성 요소 | 설명 |
|-----------|------|
| `n * 12000` | 양꼬치 n인분 총 비용 |
| `k * 2000` | 음료수 k개 전체 비용 |
| `(n/10) * 2000` | 서비스 음료수 차감 금액 |
| `n/10` | 정수 나눗셈으로 자동 floor 처리 (소수점 버림) |

---

## ❌ 틀린 코드와 비교

```cpp
// ❌ 잘못된 코드 - 음료수 수량 기준으로 서비스 계산 (오류!)
answer = n * 12000 + k * 2000 - (k/10) * 2000;

// ✅ 올바른 코드 - 양꼬치 인분 기준으로 서비스 계산
answer = n * 12000 + k * 2000 - (n/10) * 2000;
```

> ⚠️ 서비스 기준은 **음료수 수량(k)**이 아닌 **양꼬치 인분 수(n)**입니다!
