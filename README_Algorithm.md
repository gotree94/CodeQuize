## 1. 개요

세 점 P₁, P₂, P₃를 동시에 지나는 원은 **외접원(Circumscribed Circle)** 이라고 하며, 유일하게 하나 존재합니다(세 점이 일직선이 아닌 경우).

```
         P₂
        /   \
       /  O  \       O = 외접원의 중심(외심, Circumcenter)
      /    r  \      r = 외접원의 반지름
    P₁---------P₃
```

**핵심 성질**
> 원의 중심 O는 세 점 중 임의의 두 점을 잇는 선분의 **수직이등분선** 위에 항상 존재한다.

---

## 2. 수학적 원리

### 2.1 수직이등분선의 성질

선분 P₁P₂의 수직이등분선 위의 모든 점은 P₁과 P₂로부터 **등거리**입니다.

```
|OP₁| = |OP₂|   →   O는 P₁P₂의 수직이등분선 위
|OP₂| = |OP₃|   →   O는 P₂P₃의 수직이등분선 위
```

따라서 두 수직이등분선의 **교점 = 외심 O** 입니다.

### 2.2 외심의 성질

```
|OP₁| = |OP₂| = |OP₃| = r
```

세 점까지의 거리가 모두 같은 점이 외심이고, 그 거리가 외접원의 반지름입니다.

---

## 3. 단계별 계산 과정

### STEP 1. 선분 연결

세 점을 순서대로 선분으로 연결합니다.

```
P₁=(x₁,y₁), P₂=(x₂,y₂), P₃=(x₃,y₃)
```

### STEP 2. 각 선분의 중점 계산

```
M₁₂ = ((x₁+x₂)/2,  (y₁+y₂)/2)   ← P₁P₂의 중점
M₂₃ = ((x₂+x₃)/2,  (y₂+y₃)/2)   ← P₂P₃의 중점
```

### STEP 3. 수직이등분선 방향벡터 계산

선분 P₁P₂의 방향벡터: `d⃗₁₂ = (x₂-x₁, y₂-y₁)`

수직이등분선의 방향벡터 (90° 회전): `n⃗₁₂ = (-(y₂-y₁), x₂-x₁)`

```
선분 방향벡터  →  수직벡터(법선벡터)
(dx, dy)      →  (-dy, dx)
```

### STEP 4. 두 수직이등분선의 교점 계산

수직이등분선 L₁ (M₁₂ 통과, 방향 n⃗₁₂):
```
L₁: M₁₂ + t·n⃗₁₂
```

수직이등분선 L₂ (M₂₃ 통과, 방향 n⃗₂₃):
```
L₂: M₂₃ + s·n⃗₂₃
```

교점을 구하면 → **외심 O = (cx, cy)**

### STEP 5. 반지름 계산

```
r = √((x₁-cx)² + (y₁-cy)²)
```

---

## 4. 예외 처리: 일직선 판별

세 점이 일직선 위에 있으면 수직이등분선이 평행하여 교점이 존재하지 않습니다.

### 판별식 D

외심 공식의 분모인 D 값으로 판별합니다:

```
D = 2·[x₁(y₂-y₃) + x₂(y₃-y₁) + x₃(y₁-y₂)]
```

| D 값 | 의미 |
|------|------|
| D ≈ 0 | 세 점이 일직선 → 원 존재하지 않음 |
| D ≠ 0 | 외접원 계산 가능 |

D는 세 점으로 이루어진 삼각형 넓이의 2배와 같습니다:
```
|D| = 2 × 삼각형 넓이
```

## 5. 공식 유도

### 5.1 행렬식을 이용한 외심 공식

세 점이 원 위에 있다는 조건:

```
(x-cx)² + (y-cy)² = r²
```

P₁, P₂, P₃를 대입하면 3개의 방정식이 생기고, 차를 구하면:

```
2(x₂-x₁)·cx + 2(y₂-y₁)·cy = x₂²-x₁² + y₂²-y₁²
2(x₃-x₂)·cx + 2(y₃-y₂)·cy = x₃²-x₂² + y₃²-y₂²
```

이를 크래머 공식으로 풀면:

```
D  = 2·[x₁(y₂-y₃) + x₂(y₃-y₁) + x₃(y₁-y₂)]

cx = [(x₁²+y₁²)(y₂-y₃) + (x₂²+y₂²)(y₃-y₁) + (x₃²+y₃²)(y₁-y₂)] / D

cy = [(x₁²+y₁²)(x₃-x₂) + (x₂²+y₂²)(x₁-x₃) + (x₃²+y₃²)(x₂-x₁)] / D
```

### 5.2 기호 정리

```
s₁ = x₁²+y₁²
s₂ = x₂²+y₂²
s₃ = x₃²+y₃²

cx = (s₁(y₂-y₃) + s₂(y₃-y₁) + s₃(y₁-y₂)) / D
cy = (s₁(x₃-x₂) + s₂(x₁-x₃) + s₃(x₂-x₁)) / D
r  = √((x₁-cx)² + (y₁-cy)²)
```

---

## 6. 계산 시각화 요소

프로그램에서 계산 과정을 단계적으로 시각화합니다:

```
[1단계] 선분 P₁P₂, P₂P₃         → 연한 회색 실선
[2단계] 중점 M₁₂, M₂₃            → 녹색 마름모(◆)
[3단계] 수직이등분선 L₁, L₂       → 주황/보라 점선 (화면 끝까지 연장)
         중점에 수직 기호(⊾) 표시
[4단계] 중심→각 점 반지름선        → 연파랑 점선
[5단계] 외접원                     → 파랑 실선 (두께 가변)
[6단계] 클릭 지점 P₁~P₃           → 빨간 채움 원 (반지름 가변)
```

## 7. C++ 구현 코드

```cpp
void CalcCircumCircle(
    double x1, double y1,
    double x2, double y2,
    double x3, double y3,
    double& cx, double& cy, double& r,
    bool& canDraw)
{
    // 분모 D 계산
    double D = 2.0 * (x1*(y2-y3) + x2*(y3-y1) + x3*(y1-y2));

    // 일직선 판별 (부동소수점 오차 고려)
    if (fabs(D) < 1e-6) {
        canDraw = false;
        return;
    }

    // s₁, s₂, s₃ 계산
    double s1 = x1*x1 + y1*y1;
    double s2 = x2*x2 + y2*y2;
    double s3 = x3*x3 + y3*y3;

    // 외심 좌표
    cx = (s1*(y2-y3) + s2*(y3-y1) + s3*(y1-y2)) / D;
    cy = (s1*(x3-x2) + s2*(x1-x3) + s3*(x2-x1)) / D;

    // 반지름
    r = sqrt((x1-cx)*(x1-cx) + (y1-cy)*(y1-cy));

    canDraw = true;
}
```

### 원을 Polyline으로 그리기 (GDI Ellipse API 미사용)

```cpp
// 삼각함수로 원 위의 점들을 계산 → Polyline으로 연결
void DrawCircleByLine(CDC* pDC, double cx, double cy,
                      double r, int segments = 360)
{
    if (r < 1.0) return;

    std::vector<POINT> pts;
    pts.reserve(segments + 1);

    for (int i = 0; i <= segments; i++) {
        double angle = 2.0 * M_PI * i / segments;
        POINT p;
        p.x = (int)(cx + r * cos(angle));
        p.y = (int)(cy + r * sin(angle));
        pts.push_back(p);
    }
    // 첫 점과 마지막 점이 같아야 닫힌 원
    pDC->Polyline(pts.data(), (int)pts.size());
}

// 두께 있는 원: 반지름을 미세하게 달리해 여러 겹 그리기
void DrawThickCircle(CDC* pDC, double cx, double cy,
                     double r, int thickness)
{
    for (int t = 0; t < thickness; t++) {
        double offset = t - thickness / 2.0;
        DrawCircleByLine(pDC, cx, cy, r + offset);
    }
}

// 속 채운 원: 반지름 0→r 까지 촘촘히 그리기
void DrawFilledCircle(CDC* pDC, double cx, double cy, double r)
{
    for (double ri = 1.0; ri <= r; ri += 0.8)
        DrawCircleByLine(pDC, cx, cy, ri, 180);
}
```

## 8. 검증 예제

### 예제 1: 정삼각형의 외접원

```
P₁ = (0,   0)
P₂ = (4,   0)
P₃ = (2,   2√3) ≈ (2, 3.464)

D  = 2·[0(0-3.464) + 4(3.464-0) + 2(0-0)]
   = 2·[0 + 13.856 + 0] = 27.712

s₁ = 0,  s₂ = 16,  s₃ = 4+12 = 16

cx = (0·(0-3.464) + 16·(3.464-0) + 16·(0-0)) / 27.712
   = (0 + 55.424 + 0) / 27.712 = 2.0

cy = (0·(2-4) + 16·(0-2) + 16·(4-2)) / 27.712
   = (0 - 32 + 32) / 27.712 = ... 

→ 외심 O = (2, 1.155)
→ r      = 2/√3 ≈ 2.309 (정삼각형 외접원 공식과 일치)
```

### 예제 2: 일직선 판별

```
P₁ = (0, 0),  P₂ = (2, 2),  P₃ = (4, 4)

D = 2·[0(2-4) + 2(4-0) + 4(0-2)]
  = 2·[0 + 8 - 8] = 0

→ D = 0 이므로 일직선 → 원 그리기 불가
```
